Entendendo a Hierarquia das Páginas HTML

O que é HTML? (Relembrando)

HTML (HyperText Markup Language) é a linguagem usada para construir páginas web. Ela funciona através de "tags" que indicam diferentes partes de uma página, como títulos, parágrafos, listas, links, imagens e muito mais.

1. O que são tags?

Tags são instruções envolvidas entre sinais de menor e maior (< >) que dizem ao navegador como exibir o conteúdo. Exemplo: <p> define um parágrafo. Obs.: Algumas tags, como o própro <p> precisam de uma tag de fechamento, formado por: </p>

2. O que é um elemento?

Um elemento é tudo que está entre uma tag de abertura e uma tag de fechamento, incluindo o conteúdo. Exemplo: <p>Seu Texto aqui</p> é um elemento.

3. O que são atributos?

Atributos fornecem informações adicionais sobre uma tag. Exemplo: <a href="www.google.com"> Google </a> onde href="www.google.com" é um atributo do tipo chave="valor" que informa para onde seremos redirecionados ao clicar no link gerado pela tag <a>.

Exemplos de tags HTML fundamentais e seus atributos 

<h1>Esse é um título de nível 1</h1> <!-- Tag para criar titulos -->
<p>Esse é um parágrafo de texto.</p> <!-- Tag para criar paragrafos -->

<a href="https://www.exemplo.com">Esse é um link</a> <!-- Tag para criar Links -->

<img src="imagem.jpg" alt="Descrição da imagem"> <!-- Tag para criar imagens -->

<!-- Tag para criar listas -->
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
  • <h1> até <h6>: definem títulos, do mais importante (h1) ao menos importante (h6).
  • <p>: define parágrafos.
  • <a>: cria um link (com o atributo href indicando para onde seremos direcionados ao clicar no Link).
  • <img>: cria uma imagem (com os atributos: src="local" onde está salvo a imagem e alt para adicionar uma descrição a imagem).
  • <ul> e <li>: criam listas não ordenadas, com itens dentro de <li>.

Essas tags podem ser combinadas e organizadas para formar a estrutura da página.

A Hierarquia Básica de uma Página HTML

A estrutura mínima de uma página HTML segue o seguinte formato:

<!DOCTYPE html>
<html>
  
  <head>
    <title>Minha Página</title>
  </head>

  <body>
    <h1>Título Principal</h1>
    <p>Um parágrafo de texto.</p>
  </body>
  
</html>

Explicando cada parte:

  • <html> é o elemento raiz de toda a página.

  • Dentro do <html>, temos dois filhos principais:

    • <head>: parte da página que contém informações como o <title>.
    • <body>: onde ficam os conteúdos visíveis ao usuário (títulos, textos, imagens, etc).

  • <title> é o título que aparece na aba do navegador.

  • <h1> é um título visível na página.

  • <p> representa um parágrafo.

Cada elemento HTML pode conter outros elementos dentro dele, formando uma estrutura de árvore. Isso é chamado de hierarquia, e é essencial para entender como os elementos se organizam e se relacionam. Veja abaixo essa esturura em fomato de caixas.


Visualizar as tags HTML como uma estrutra de caixa que pode conter outras caixas (tags HTML) é fundamental para entender como as paginas HTML funcionam. Observe que cada tag HTML (que a partir de agora iremos chamar também de caixa) possui outras caixas dentro. Exemplo: <body> possui como filhos: <h1> e <p>.

Conclusão

Entender a hierarquia do HTML ajuda a construir páginas bem organizadas e a usar o CSS de forma eficaz. Com o tempo, você poderá criar estruturas mais complexas e controlar completamente o visual do seu site.
